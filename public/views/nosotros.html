Bootcamp Full Stack Engineer

Este trabajo es el Proyecto Integrador del Bootcamp Full Stack Engineer realizado en Educación IT.

Dictado desde el 15/03/2022 al 03/11/2022 por el Profesor Juan Domingo Romero en modalidad en vivo online de 2 clases por semana de 3 hs c/u (Total 198 hs).

Todos los conocimientos adquiridos a lo largo del bootcamp están orientados al desarrollo de soluciones de Arquitectura WEB tanto de Front End como Back End y se basan en estos tres pilares:

HTML (HyperText Markup Language), CSS (Cascading Style Sheets) y JavaScript Vanilla.

Para el Front End aprendimos a usar derivados, complementos y herramientas auxiliares tales como SASS (Syntactically Awesome Stylesheets), la especificación ECMA script, el manejo de código asincrónico con AJAX (Asynchronous JavaScript And XML), Handlebars y SPA (Single Page Aplication).

En el Back End usamos node.js para crear servidores http e implementamos varias librerías como Express y MongoDb entre otras.
